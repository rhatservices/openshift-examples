* A full example for exposing an applictation

  In this example we are going to deploy and expose nginx.

** Example 1: Deploy and expose nginx

   #+begin_src sh
oc apply -f nginx-deployment.yml
oc apply -f nginx-service.yml
oc expose nginx
   #+end_src
